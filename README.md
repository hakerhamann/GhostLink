# GhostLink

GhostLink — защищенный приватный мессенджер для узкого круга лиц.

Приложение создано для личного общения в закрытом пространстве: без публичной регистрации, лишнего шума и открытой социальной ленты. В центре проекта — прямые диалоги, групповые чаты, медиа-сообщения, уведомления и контролируемое распространение обновлений.

Клиент GhostLink по умолчанию работает с удаленным сервером проекта. Локальный запуск backend не требуется для обычного использования приложения.
