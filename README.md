# Резерв (локальный мессенджер)

Проект теперь работает без Firebase: backend запускается локально на вашем ПК.

## Что уже реализовано

- Регистрация и вход по логину/паролю
- Сессии по токену
- Список диалогов
- Прямой чат между двумя пользователями
- Отправка сообщений и отметка прочитанного
- Тёмный UI в стиле вашего макета

## 1) Запустить backend на ПК

```powershell
cd C:\Projekt\APK\backend
.\start_backend.ps1 -Port 8080
```

Проверка:

```powershell
Invoke-WebRequest http://127.0.0.1:8080/health | Select-Object -ExpandProperty Content
```

## 2) Узнать IP вашего ПК в Wi-Fi

```powershell
ipconfig
```

Нужен IPv4 адрес активного адаптера, например `192.168.1.45`.

## 3) Собрать APK

```powershell
cd C:\Projekt\APK
.\gradlew.bat assembleDebug
```

APK:

`C:\Projekt\APK\app\build\outputs\apk\debug\app-debug.apk`

## 4) Что вводить в приложении

На экране входа есть поле адреса сервера.

- Если это эмулятор Android Studio: `http://10.0.2.2:8080`
- Если это реальный телефон в той же Wi-Fi сети: `http://<IP_ПК>:8080`

Пример: `http://192.168.1.45:8080`

## 5) Проверка переписки между двумя людьми

1. Установить APK на два телефона (или два инстанса).
2. На первом: зарегистрировать пользователя `user1`.
3. На втором: зарегистрировать пользователя `user2`.
4. В `user1` нажать кнопку с карандашом и ввести логин `user2`.
5. Откроется диалог — отправка/получение сообщений начнет работать.
